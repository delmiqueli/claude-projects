🤖 What is Claude Code?

Claude Code is an AI coding assistant made by Anthropic — the same company behind the Claude chatbot you may have used on claude.ai.

Think of it like having a very smart helper who lives inside your computer. You type what you want in plain English — "build me a simple website" or "fix this bug" — and Claude does the heavy lifting.

Unlike claude.ai (a website), Claude Code runs as a tool on your Mac. It can see your files, write code, and make changes — with your permission at every step.

Phase 1 — Setup
Installing the Tools
01

Download & Install Node.js

Claude Code needs a helper program called Node.js to run on your Mac.

Node.js is free software that lets programs like Claude Code run on your computer. Think of it as a required "ingredient" — once installed, you never need to touch it again.

  1. Open any web browser (Safari, Chrome, or Firefox) on your Mac.
  2. Go to nodejs.org — type this in your address bar and press Return.
  3. Click the green button labeled "LTS" (Long Term Support — this is the most stable version).
  4. A file downloads to your Downloads folder. Double-click it to open the installer.
  5. Click through: Continue → Continue → Install. Enter your Mac password if asked.
  6. When you see "The installation was successful" — click Close. Done! ✅

02

Open the Terminal App

Terminal is a text-based window that comes pre-installed on every Mac.

Terminal lets you give your computer instructions by typing commands. It looks simple and plain — and that's okay! You only need to follow exactly what this guide shows you.

  1. Press ⌘ Command + Space at the same time. A search bar appears in the center of your screen — that's Spotlight.
  2. Type Terminal and press Return.
  3. A window opens — mostly white or black with a blinking cursor. That's Terminal. You're in!

03

Install Claude Code

One command is all it takes — Claude Code installs itself automatically.

Click inside the Terminal window to make sure it's active. Then carefully type (or copy and paste) the command below, and press Return:

Terminal — zsh
~ $ npm install -g @anthropic-ai/claude-code

added 47 packages in 12s
✓ claude-code installed successfully
Run 'claude' to get started

~ $

Text will scroll by while it installs (30–90 seconds). You'll know it's done when the scrolling stops and your prompt (~ $) reappears.

📋
Copy-paste tip To paste into Terminal: ⌘ Command + V — same as anywhere on your Mac.
⚠️
If you see "permission denied" Type sudo npm install -g @anthropic-ai/claude-code instead, press Return, then type your Mac password (it won't show as you type — that's normal).

Phase 2 — Connect
Signing In & Opening Claude Code
04

Launch Claude Code & Sign In

Connect Claude Code to your Anthropic account to activate it.

Claude Code needs to connect to Anthropic's servers. You'll sign in just like you do on claude.ai. Here are the two ways to access it:

✅ Claude Pro or Max Plan
If you pay for Claude Pro or Max on claude.ai, Claude Code is already included. Sign in with the same account.
🔑 API Key (Pay as you go)
Use your Anthropic API key from console.anthropic.com. You'll be billed based on usage.

In Terminal, type the following and press Return:

Terminal — zsh
~ $ claude

Welcome to Claude Code!
Opening your browser to sign in...
  1. Type claude and press Return.
  2. Your web browser opens with a login page — sign in with your Anthropic account.
  3. Click "Authorize" when prompted.
  4. Your browser may show "You can return to Claude Code." Switch back to Terminal.
  5. Claude Code is now ready and waiting for your first message! 🎉
💡
Don't have an account yet? Go to claude.ai and sign up first. Claude Code requires at least a Claude Pro subscription to use.
05

Open the Claude Code Desktop App

After installing, Claude Code gives you a proper Mac app — no Terminal needed from here on.

The installation you did in Phase 1 also installed the Claude Code desktop app. This is a clean, visual interface — much friendlier than Terminal — and it's the recommended way for beginners to use Claude Code day-to-day.

  1. Press ⌘ Command + Space to open Spotlight Search.
  2. Type Claude Code and press Return.
  3. The app opens. You'll see three tabs at the top: Chat, Cowork, and Code. Click Code — that's the main coding interface.
  4. Sign in with your Anthropic account if prompted (same login as claude.ai).

Phase 3 — Use It!
Talking to Claude Code
06

Select Your Project Folder

Use the "Select folder" button to point Claude at the folder you want to work in — no Terminal needed.

Before you can ask Claude to do anything, it needs to know where your project lives on your Mac. The desktop app makes this easy with a simple folder picker.

  1. In the Claude Code app, look for the "Select folder" button near the bottom-left of the input area.
  2. Click it. A standard Mac file browser opens.
  3. Navigate to your project folder — or your Desktop — and click Open.
  4. The button updates to show your folder name, confirming Claude is now looking at those files.

🖥️
What does "Local" mean? The "Local" button in the bottom-right of the input area means Claude Code is running on your own Mac. You can leave this as-is — it's the right setting for most people.
07

Type Your First Prompt & Review the Results

Ask Claude anything in plain English, then review, approve, and keep refining.

Now the real magic happens. Click inside the text box, type what you want, and press the orange send button (↑) or hit Return. Claude Code understands plain English — no coding knowledge needed.

← You're always in control

Claude shows you exactly what it wants to do — and waits for your go-ahead before touching any files. Click Approve to let it proceed, or Reject to say no. You can also keep chatting to refine the result.

  1. After approving, open your project folder in Finder — new files will appear there immediately.
  2. For HTML files: double-click to open in Safari and preview your webpage!
  3. Want changes? Type follow-up instructions like "Change the background to dark green" or "Make the text bigger."
  4. Claude proposes each change and waits for your approval. Keep going until you're happy!
🖥️
When Claude asks "Which dev server would you like to start?" Always choose option 1 — "Python Static Server" (it says "built-in, no install needed"). The Preview panel on the right will then show your live webpage. Avoid option 2 (npx serve) for now — it can trigger a permissions error (a wall of red text starting with npm error EACCES). If that happens, your HTML file was still created — just find it in Finder and double-click to open in Safari.
🔧
Fixing the "npm error EACCES" permissions error If Claude tells you to run a sudo chown command: open Terminal (⌘ + Space, type Terminal, press Return), paste the command Claude gives you — it looks like sudo chown -R 502:20 "/Users/yourname/.npm". Press Return, type your Mac password when asked (it won't show as you type — that's normal), then go back to Claude Code and type "done." Claude will restart the server. This is a one-time fix.
💡
The more specific, the better Instead of "make a website," try: "Make a website for my bakery called Sunny Loaves with a menu section showing croissants, muffins, and sourdough. Use warm golden colors and a clean, modern look."
🔄
Leave "Auto accept edits" off while learning When on, Claude makes changes without asking each time. Leave it off while you're getting started so you can review every change before it happens.
📝
What are the quick-action cards? The suggestion cards on the home screen ("Create or update my CLAUDE.MD file", "Search for a TODO comment…") are clickable shortcuts. The CLAUDE.MD one is especially useful — it creates a memory file so Claude remembers your project details between sessions.
